Installing

psn-php can be installed using Composer. It's as easy as running this command:

composer require tustin/psn-php
or if you want to use the dev branch: 
composer require tustin/psn-php=dev-master

Composer will download all the files required for the API wrapper.

Setting Up

Once all the files have been downloaded, you can include the API into your project. Require the Composer autoloader at the beginning of your script(s): 

require "vendor/autoload.php"

You now have access to the API wrapper.
