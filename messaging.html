Setting Up

Use the array of tokens you obtained, either from GetTokens or GrabNewTokens, to create a new instance of the Messaging class:

$messaging = new \PSN\Messaging($tokens);

You can now use $messaging to make requests related to the logged in user's messages.

GetAll()

GetAll takes no arguments and returns an object of all groups you're currently active in:

$my_groups = $messaging->GetAll();
//you can use var_dump to see the contents of this object
var_dump($my_groups);

GetFavorites()

GetFavorites takes no arguments and returns an object of all your favorited groups:

$my_favorites = $messaging->GetFavorites();
//you can use var_dump to see the contents of this object
var_dump($my_favorites);

Remove($MessageGroupID)

Remove takes one argument, which is the ID of the messaging group you want to delete:

$messaging->Remove('1111111');

FavoriteGroup($MessageGroupID)

FavoriteGroup takes one argument, which is the ID of the messaging group you want to favorite:

$messaging->FavoriteGroup('1111111');

UnfavoriteGroup($MessageGroupID)

UnfavoriteGroup takes one argument, which is the ID of the messaging group you want to unfavorite:

$messaging->UnfavoriteGroup('1111111');

GroupAddUsers($MessageGroupID, $PSN)

GroupAddUsers takes two arguments:

  • $MessageGroupID - ID of the group you want to add user(s) to
  • $PSN - User(s) you want to add to the group. This argument can either be an array of users OR a single PSN as a string

$messaging->GroupAddUsers('1111111', "foobar");
//or
$messaging->GroupAddUsers('1111111', ["foo", "bar"]);

AddGroupImage($MessageGroupID, $Image)

AddGroupImage takes two arguments, the ID of the group you want to add an image to, and the URL of the image or the image contents itself:

$messaging->AddGroupImage('1111111', 'https://static.independent.co.uk/s3fs-public/thumbnails/image/2014/09/12/22/Gary_Busey.jpg');

RemoveGroupImage($MessageGroupID)

AddGroupImage takes one argument which is the ID of the group you want to remove the image from:

$messaging->RemoveGroupImage('1111111');

SetGroupName($MessageGroupID, $Name)

SetGroupName takes two arguments, the ID of the group you want to change the name of and the new name for the group:

$messaging->RemoveGroupImage('1111111', 'bolshoi booze');

Get($MessageGroupID)

Get takes one argument which is the ID of the group you want to get the messages from:

$messaging->Get('1111111');

GetAudioAttachment($URL)

GetAudioAttachment takes one argument which is the URL of the audio:

$messaging->GetAudioAttachment('some-audio-url.mp3');

GetImageAttachment($URL)

GetImageAttachment takes one argument which is the URL of the image:

$messaging->GetImageAttachment('some-image-url.jpg');

AudioMessage($PSN, $Audio, $AudioLength)

AudioMessage takes three arguments:

  • $PSN - User(s) you want to send the audio message to. This argument can either be an array of users OR a single PSN as a string
  • $Audio - URL or raw data of the audio
  • $AudioLength - How long the audio is (presumed seconds)

$messaging->AudioMessage('foobar', 'my-audio-link.mp3', 10);

AudioMessageGroup($GroupID, $Audio, $AudioLength)

AudioMessageGroup takes three arguments:

  • $GroupID - ID of the group you want to send the audio message to
  • $Audio - URL or raw data of the audio
  • $AudioLength - How long the audio is (presumed seconds)

$messaging->AudioMessage('1111111', 'my-audio-link.mp3', 10);

ImageMessage($PSN, $Image [, $Message = ""])

ImageMessage takes three arguments:

  • $PSN - User(s) you want to send the image to. This argument can either be an array of users OR a single PSN as a string
  • $Image - URL or raw data of the image
  • $Message - Optional argument. Some text sent along with the image

$messaging->ImageMessage('foobar', 'my-image-link.jpg');
//or
$messaging->ImageMessage(['foo', 'bar'], 'my-image-link.jpg');

ImageMessageGroup($GroupID, $Image [, $Message = ""])

ImageMessageGroup takes three arguments:

  • $GroupID - ID of the group you want to send the image to
  • $Image - URL or raw data of the image
  • $Message - Optional argument. Some text sent along with the image

$messaging->ImageMessageGroup('1111111', 'my-image-link.jpg');

TextMessage($PSN, $Message)

TextMessage takes two arguments:

  • $PSN - User(s) you want to send the message to. This argument can either be an array of users OR a single PSN as a string
  • $Message - Message to send to the user(s)

$messaging->TextMessage('foobar', 'its only smellz');
//or
$messaging->TextMessage(['foo', 'bar'], 'its only smellz');

TextMessageGroup($GroupID, $Message)

TextMessageGroup takes two arguments:

  • $GroupID - ID of the group you want to send the image to
  • $Message - Message to send to the group

$messaging->TextMessageGroup('1111111', 'its only smellz');
