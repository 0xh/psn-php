Setting Up

Use the array of tokens you obtained, either from GetTokens or GrabNewTokens, to create a new instance of the Trophy class:

$trophy = new \PSN\Trophy($tokens);

You can now use $trophy to make requests related to the logged in user's trophies.

GetMyTrophies([ $Limit = 36 ])

GetMyTrophies takes no arguments and will return an object with multiple arrays containing all your trophies and information regarding them. There is an optional argument $Limit that will limit how many trophies get returned:

$my_trophies = $trophy->GetMyTrophies();
//you can use var_dump to see the contents of this object
var_dump($my_trophies);

GetGameTrophies($GameID)

GetGameTrophies takes one argument, which is the ID for the game you want to get the trophies for. A normal game ID will look something like NPWR07466_00, which is Rocket League:

$game_trophies = $trophy->GetGameTrophies("NPWR07466_00");
//you can use var_dump to see the contents of this object
var_dump($game_trophies);
